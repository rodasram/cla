== Description ==
Sports Lite is a dynamic and colorful, creative and responsive, lightweight and flexible, sleek and smooth, powerful and engaging sports academy WordPress theme for sports professionals. It is a perfect platform to create a elegant website for sports club, sports blogs, sports magazines and all similar sports related websites.
